About This Project

What is TI-DE?

🎯
TTP-Based Detection Only
No hardcoded IPs or domains. Attacker infrastructure rotates in hours — adversary TTPs persist for years. Every rule targets behavior, not indicators.
🔵
11 Security Platforms
Each report ships detection rules for SIEM and EDR/XDR platforms: Sigma, Sentinel, Splunk, QRadar, Defender, Cortex XDR, CrowdStrike (SPL + IOA), Carbon Black, SentinelOne, Kaspersky, YARA.
⚡
Published Within 24 Hours
CISA KEV entries and actively exploited CVEs are processed within 24 hours of confirmed exploitation intelligence. Speed matters in the detection gap window.
🔴
Red Team → Blue Team Flow
Every report follows attacker perspective first: attack chain, live simulation, MITRE timeline with forensic evidence. Then blue team: SIEM rules, EDR rules, response playbook.
🌍
Open Source · TLP:WHITE
All detection rules are freely available. TLP:WHITE — no paywalls, no registration. The security community grows stronger when detection logic is shared openly.
📖
Bilingual Reports (EN / TR)
Every report is fully available in English and Turkish. Toggle language at any time — content, labels, and navigation switch instantly without page reload.
